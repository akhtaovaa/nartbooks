# Как запустить NartBooks

## Быстрый запуск

### Вариант 1: Через скрипты (рекомендуется)

**Windows (CMD):**
1. Дважды кликни на `start_backend.bat` - запустится backend
2. Дважды кликни на `start_frontend.bat` - запустится frontend сервер
3. Открой в браузере: http://localhost:8080

**Windows (PowerShell):**
1. Правой кнопкой на `start_backend.ps1` → "Выполнить с PowerShell"
2. Правой кнопкой на `start_frontend.bat` → "Открыть"
3. Открой в браузере: http://localhost:8080

### Вариант 2: Вручную через терминал

**Backend:**
```bash
# Открой терминал в корне проекта
python -m uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

**Frontend:**
```bash
# В другом терминале, перейди в папку frontend
cd frontend
python -m http.server 8080
```

Затем открой в браузере: http://localhost:8080

## Проверка работы

1. **Backend API:** http://localhost:8000/docs
2. **Frontend:** http://localhost:8080

## Если не запускается

1. **Проверь Python:**
   ```bash
   python --version
   ```

2. **Установи зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Проверь, что ты в правильной директории** (там где есть папки `app/` и `frontend/`)

4. **Если ошибка "ModuleNotFoundError: No module named 'app'":**
   - Убедись, что запускаешь из корня проекта (где есть `app.py`)
   - Или используй скрипты `start_backend.bat` / `start_backend.ps1`

## Порты

- **Backend:** 8000 (http://localhost:8000)
- **Frontend:** 8080 (http://localhost:8080)

Если порты заняты, измени их в скриптах запуска.
